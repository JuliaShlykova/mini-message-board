# Mini message board
  To see the website visit [mini-message-board](https://mini-message-board-rjo6.onrender.com).
## Description
  This project is the part of [The Odin Project](https://www.theodinproject.com/). It represents a simple messenger.
## Technologies used
  - NodeJS
  - Express library
  - MongoDB database
## MongoDB database
There is a simple collection of "messages", a document of which has a structure: 
```
{
  username: String,
  text: String,
  date_added: Date
}
```
